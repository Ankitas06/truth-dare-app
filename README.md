# 🎲 Truth or Dare Web Application

An interactive multiplayer **Truth or Dare** web application built using **Node.js, Express.js, and Socket.IO**. The application enables users to play the classic Truth or Dare game in real time through a responsive and engaging interface.

---

## 🚀 Features

* 🎯 Random Truth and Dare questions
* 👥 Multiplayer gameplay using Socket.IO
* ⚡ Real-time communication between players
* 📱 Responsive and user-friendly interface
* 🔄 Dynamic question generation
* 🎨 Clean and interactive UI

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Ankitas06/truth-dare-app.git
```

### Navigate to the project folder

```bash
cd truth-dare-app
```

### Install dependencies

```bash
npm install
```

### Start the application

```bash
npm start
```

The application will run on:

```text
http://localhost:3000
```

---

## 💡 How It Works

1. Launch the application.
2. Players join the game.
3. The system randomly selects either **Truth** or **Dare**.
4. A random prompt is displayed.
5. Continue playing with multiple participants in real time.
